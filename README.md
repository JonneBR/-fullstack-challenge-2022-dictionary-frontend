## Getting Started

### Prerequisites

- yarn >=1.22.17
- node >=16.0.0

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/JonneBR/fullstack-challenge-2022-dictionary-backend
   ```
2. Install packages
   ```sh
   yarn
   ```
3. Run tests
   ```sh
   yarn test
   ```
4. Run the app locally
   ```sh
   yarn dev
   ```
